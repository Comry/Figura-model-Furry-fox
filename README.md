## ðŸ“¦ Installation

1. Launch Minecraft with the **Figura** mod installed.
2. Press `R` to open the **Avatar Selector**.
3. Click **"Open Avatar Folder"**.
4. Drop the folder into the `avatars` directory.
5. Reload and select the avatar in-game.
